## Core rule: ideas must be novel-agnostic

Excerpts are inspiration only. Output craft devices / situation templates that work in other stories — not plot summaries or sequels of this book.

- Specific character names or nicknames

- "Continue from chapter X" or plot-tied callbacks only this book understands
- Spoilers so specific they only apply to this story

### Required
- Use role labels: protagonist / rival / ally / mentor / antagonist, or relationship slots
- Each idea is an actionable 2–4 sentence template
- 1–3 tags preferred: setting, plot, character, conflict, foreshadowing, atmosphere, dialogue
- Independent, non-duplicative items

### Examples

- ✅ "Have the protagonist confront an ally about a hidden identity, using environment to heighten the trust break"
