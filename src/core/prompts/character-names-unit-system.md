## 目标
1. 找出所有**具体人物**的指称：真名、外号、封号、有辨识度的职务称呼（如「周总」「许老师」）。  
2. **本窗局部消解**：本窗内已能判定为同一人的称呼，**必须合成一条**（`name` + `aliases`）。  
3. **不做跨窗/全书合并**。  
4. 只输出 `name` / `aliases` 字符串（位置由程序记窗）。

## name 是什么（硬规则）
- **`name` 必须是可独立指称的标签**：真名优先；没有真名时用外号/封号/「姓+职务」等**仍能指到固定个体**的叫法。  
- **禁止**把**悬空指代 / 悬空关系称谓**单独做成一行的 `name`。

### 禁止当 name（悬空 / 依赖说话人视角）
| 类型 | 例 | 正确做法 |
|------|-----|----------|
| 纯代词/泛指 | 他、她、我、你、有人、众人 | **不列** |
| 指示亲属 | 他爸、我妈、你哥 | 能落到本窗具体人 → 进其 **aliases**；落不到 → **不列** |
| 无主关系称谓 | 小儿子、女朋友、弟弟、父亲、母亲、老公、老婆（未绑到本窗姓名/外号） | 本窗已有「周屿」等 → **aliases**；单独出现 → **不列** |
| 悬空职务 | 单独「老师」「总」无姓无名 | **不列**（「许老师」「周总」可以） |

### 允许当 name
- 真名：周屿、许栀、孙悟空  
- 稳定外号/封号：齐天大圣、黑仔、平头大叔  
- 有名号职务：周总、许老师  

示例：
- 本窗「周屿」又被叫「小儿子」→ `name=周屿`, `aliases=["小儿子",…]`  
- 本窗只有「他女朋友」且未出现姓名 → **不输出**这一人（勿 `name=女朋友`）  
- 「孙悟空」+「齐天大圣」→ `name=孙悟空`, `aliases=["齐天大圣"]`  
- 仅有「齐天大圣」→ `name=齐天大圣`, `aliases=[]`（允许）

## 必须排除
1. 纯代词 / 泛指  
2. 悬空亲属与悬空关系称谓（见上表）  
3. 非人、无稳定个体  

## 规则
1. **一人一行**。`name` = 本窗最佳**非悬空**称呼（真名优先）。`aliases` = 同人其它写法（可含关系称谓）。  
2. **禁止**把本窗已判定为同一人的真名与封号拆成两行。  
3. **禁止编造**姓名；禁止用悬空关系词凑一行。  
4. **禁止跨窗全书消解**。  
5. 不要输出性格、剧情、关系网——只要指称结构。
