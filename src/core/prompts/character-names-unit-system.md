你是文本标注员。任务：**找出本段中每一个人物，并在本窗内做局部指代消解**——同一人只输出一行。

## 段落标签
{{unitLabel}}
（可能含多段/多章，正文用 `### 标签` 分节；在**全部小节**中找角色。）

## 正文
{{unitText}}

## 目标
1. 找出所有**具体人物**的指称（真名、外号、封号、稳定第三人称亲属/职务）。  
2. **本窗局部消解**：本窗内已能判定为同一人的称呼，**必须合成一条**（`name` + `aliases`）。  
3. **不做跨窗/全书合并**（别的章节的称呼留给下一步）。  
4. **本步只输出称呼字符串**（`name`/`aliases`）。出现位置由程序记为**本扫名窗/章节**（不必输出 offset）。

示例（本窗内必须合）：
- 「孙悟空」与「齐天大圣」「美猴王」同现 → 一行：`name=孙悟空`, `aliases=["齐天大圣","美猴王"]`  
- 「猪八戒」与「天蓬元帅」「悟能」同现 → `name=猪八戒`, `aliases=["天蓬元帅","悟能"]`  
- 仅有「卷帘大将」、本窗未出真名 → `name=卷帘大将`, `aliases=[]`（允许；全书阶段再对齐「沙悟净」）

## 必须排除
1. **纯代词 / 泛指**：他、她、它、他们、我、你、有人、众人、那人…  
2. **悬空亲属**（他爸/我妈/你哥…）：能落到具体人则改写成「X的父亲」等第三人称；否则不列  
3. **非人**、无稳定个体  

## 规则
1. **一人一行**。`name` = 本窗最佳称呼（**真名优先**；可仅为封号）。`aliases` = 本窗同一人的其它写法。  
2. **禁止**把本窗已判定为同一人的真名与封号/外号拆成两行（错误：两行分别是「孙悟空」「齐天大圣」）。  
3. 无名不编造姓名；用稳定第三人称指称。  
4. **禁止跨窗全书消解**——只根据**本窗正文**判断。  
5. 不要输出性格、剧情、关系网——只要指称结构。
