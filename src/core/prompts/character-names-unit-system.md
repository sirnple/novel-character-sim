## 目标（第一步 = 找角色，不是找专名）
许多角色**没有姓名**，正文只用指代/描述称呼他们。这些也必须列出，例如：
- 亲属指代（**第三人称稳定说法**）：「周屿的母亲」「周伯彦的妻子」「周屿的父亲」
- 外貌/外号：「短发大叔」「黑仔」「果园老伯」
- 职务/身份指代：「教导主任」「周总」（指向具体人物时）
- 正式姓名：「周屿」「许栀」

**禁止**因为「没有姓名」就漏掉已故亲属、路人配角、只用描述出场的人。

## 必须排除（模型自行过滤，勿输出）
这些**不是**可收录的角色 surface：

1. **纯代词 / 泛指**：他、她、它、牠、他们、她们、它们、我、你、您、咱、俺、自己、人家、某人、有人、众人、大家、那人、那家伙  
2. **悬空亲属/关系词**（说话人视角、无法单独当角色名）：他爸、他妈、她爸、她妈、他爹、他娘、他哥、他姐、他弟、他妹、他老婆、她老公、他儿子、她女儿、我爸、你妈、您父亲 等  
   - 若本段能确定具体人：改写成稳定第三人称再列（如「周屿的父亲」），**不要**把「他爸」原样输出  
   - 若无法落到具体人：**整条不列**  
3. **非人**：动物用「它」、物品、地名、组织、功法、纯称谓无个体（「老师」若无具体指代）  
4. **无稳定个体**：「有人」「众人」「那人」

## 规则
1. 每条输出一个 **surface**。`name` 填该 surface；`aliases` 填本段内同一人的其它写法。  
2. **有名用名，无名用第三人称稳定指称**：如「周屿的母亲」「短发大叔」；**不要编造姓名**。  
3. 收录：真名、外号、封号、简称、稳定描述称呼、**已能锁定个体**的亲属/职务称谓。  
4. **本步不做全书指代消解**：同段「孙悟空」与「齐天大圣」可都列。  
5. 在「是否具体人物」拿不准时：若已是姓名/外号/稳定第三人称 → 倾向列入；若只是代词/他爸类 → **排除**。  
6. 不要输出性格、剧情、关系网——只要指称字符串列表。
