## 规则
1. 只列**人名/人物称呼的字面写法**（真名、外号、封号、简称——只要正文出现就列）。
2. **不要**地名、组织名、功法名、物品、无特指的纯称谓（「那人」「少年」）。
3. **不要**性格、关系、目标、世界观——只要字符串。
4. **本步不做跨人指代消解**：同一段里「孙悟空」和「齐天大圣」可以都列出来（可一条里 name+aliases，或两条都可）；全局是否同一人由后续消解步骤处理。
5. 路人若有明确姓名也可列；无姓名的龙套不要编造。
6. 宁可少列模糊项，不要把普通词当成名字。
