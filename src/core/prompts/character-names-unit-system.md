## 目标（第一步 = 找角色，不是找专名）
许多角色**没有姓名**，正文只用指代/描述称呼他们。这些也必须列出，例如：
- 亲属指代（优先**第三人称稳定说法**）：「周屿的母亲」「周伯彦的妻子」；对话里的「他爸」若能对应具体人也可列
- 外貌/外号：「短发大叔」「黑仔」「果园老伯」
- 职务/身份指代：「教导主任」「周总」（指向具体人物时）
- 正式姓名：「周屿」「许栀」

**禁止**因为「没有姓名」就漏掉已故亲属、路人配角、只用描述出场的人。

## 规则
1. 每条输出一个 **surface**（正文中的字面指称串）。可用 `name` 字段填该 surface；`aliases` 填本段内同一人的其它写法。
2. **有名用名，无名用第三人称指称**：没有真名时，`name` 用最稳定第三人称（如「周屿的母亲」「短发大叔」），**不要编造姓名**。
3. **aliases 不要写第一/二人称**：「我爸」「你妈」「您父亲」取决于说话人，后续名单里不会当正式别名；若正文只有这类说法，优先改写成可识别的第三人称（有依据时），否则可单列 surface 但别当 canonical name。
4. 收录：真名、外号、封号、简称、稳定描述称呼、指向特定个体的亲属/职务称谓。
5. **不收录**：地名、组织、物品、功法；完全无特指的「有人」「众人」「那人」。
6. **本步不做全书指代消解**：同一段里「孙悟空」和「齐天大圣」可都列；全局是否同一人由后续步骤处理。
7. **高召回优先**：拿不准但是否「在说某个具体的人」时，倾向列入。
8. 不要输出性格、剧情、关系网——只要指称字符串列表。
